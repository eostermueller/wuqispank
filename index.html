What is wuqiSpank?

wuqiSpank is a web app that traces and visualizes your app's JDBC activity

What is a wuqi?

A wuqi is an Unecessary Query Invocation.

Why would I want a spank a wuqi?

Everyone knows that wuqis are the number one performance problem today.

3rd Party Dependencies

This project sits on the shoulders of mxGraph/JGraph, InTrace and Wicket.
